## 异常

1. Example:

    ```Python
    try:
         测试语句 
         except 异常名称， 异常数据:
         处理异常语句 
         except 异常名称， 异常数据:
         处理异常语句 
         else:
         未出现异常执行语句 
         finally:
         不管有没有异常都需要执行的语句 
    ```
